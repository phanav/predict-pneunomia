Evaluation

The problem is a binary classification problem, and you are given train set and train solutions set for training. You have to build a model to label every row of the validation set and the test set with a 0 or a 1, corresponding to the two classes "Normal" or "Pneumonia".  You will receive feed-back on your performance on the validation set since we provide you the validation solutions set but there's no test solutions set. You need to upload your predictions of test set and your performance will display on the leaderboard.

The submissions are evaluated by precision.

Precision = True Positive / All Predicted Positive
